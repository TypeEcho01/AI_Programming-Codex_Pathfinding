# BFS + DFS Pathfinding (Python Console)

This project supports two modes:
- **Pathfinding demos** on two built-in maps (prints found/path length/visited + overlays)
- **Monster Chase** turn-based game (`P` player vs `M` monster, with `MODE` set to BFS/DFS)
Runs two built-in maps and prints:
- whether a path was found
- path length
- visited count
- rendered map overlays for visited/path tiles

## Run
```bash
python pathfinding.py
```

At startup, choose:
- `1` for pathfinding demos
- `2` for Monster Chase
